SPRINT 1:
1. As a user, I want to play, pause and stop songs so that I can control my music easily.
2. As a user, I want to skip to the next or previous song so that I can choose what I listen to.
3. As a user, I want to see the current song title, artist name and duration so that I know what is playing.
4. As a user, I want to change the volume between 0 and 100 so that I can set it to a comfortable level.
5. As a user, I want to see the song progress and total time so that I can tell how long it has left.
6. As a user, I want to see a progress bar and be able to type a time to jump to in the song so that I can move to a specific point.
7. As a user, I want to use keyboard shortcuts for play, pause, stop and mute so that I can control playback quickly.
8. As a user, I want to rewind or fast-forward by five seconds so that I can skip small parts easily.
9. As a user, I want to mute and unmute the sound instantly so that I can quickly silence the music.
10. As a user, I want an indicator to show which song is playing so that it is easy to see in a list.
11. As a user, I want a list of help commands so that I know how to navigate the application.
12. As a user, I want the player to keep running in the background so that music continues while I use other commands.
SPRINT 2:
1. As a user, I want to create, rename and delete playlists so that I can organise my music library easily.
2. As a user, I want to add, remove and change the order of songs in a playlist so that I can choose the order of tracks.
3. As a user, I want to search for songs by title, artist or file name so that I can find specific tracks quickly.
4. As a user, I want to view songs, albums and artists in a clear text table so that the information is easy to read.
5. As a user, I want to see a list of all my playlists and pick one by typing its name or number so that I can switch between them quickly.
6. As a user, I want to open a playlist and see all its songs with numbers and durations so that I can view its contents easily.
7. As a user, I want to add a song from my main library to a playlist using a text command so that I can organise tracks faster.
8. As a user, I want a confirmation message when I add or remove a song from a playlist so that I know the change worked.
9. As a user, I want the player to scan my music folder for new files so that my library stays up to date automatically.
10. As a user, I want to see the number of songs and total play time for each playlist so that I can see how long it is.
11. As a user, I want to merge two playlists and choose whether to remove duplicate songs so that I can combine them easily.
12. As a user, I want to make a copy of a playlist so that I can edit it without changing the original.
SPRINT 3:
1. As a user, I want to turn shuffle mode on or off so that songs play in a random order when I choose.
2. As a user, I want to turn loop mode on for one song or all songs so that tracks can repeat automatically.
3. As a user, I want to see the current queue and recently played songs as text so that I know what is coming next.
4. As a user, I want to add or remove songs from the queue so that I can control what plays next.
5. As a user, I want to queue a specific song to play next so that I can hear it straight after the current one.
6. As a user, I want to clear the whole queue with one command so that I can start a new list easily.
7. As a user, I want to change the playback speed between 0.5× and2× so that I can listen faster or slower.
8. As a user, I want to like or unlike songs so that I can mark my favourites.
9. As a user, I want to view my liked songs in a single list so that I can play them quickly.
10. As a user, I want to sort playlists automatically by artist, title or duration so that they are organised better.
11. As a user, I want to see a list of my most‑played songs so that I can find my favourites easily.
12. As a user, I want to set a sleep timer so that the music stops after a chosen time.
SPRINT 4:
1. As a user, I want the player to remember my last settings such as volume, shuffle and loop so that it starts how I left it.
2. As a user, I want to schedule playback at certain times so that music starts automatically during my routine.
3. As a user, I want the player to remember the last song and position when I close it so that I can continue later.
4. As a user, I want to import songs from local or external folders so that I can add new music easily.
5. As a user, I want to create custom tags or labels for songs and filter by them so that I can group music by mood.
6. As a user, I want to see a “Recently Added” list so that I can find new songs quickly.
7. As a user, I want to create and switch between multiple user profiles so that each listener can maintain separate preferences and playlists.
8. As a user, I want to view detailed playback statistics (total listening time, most-played artists) so that I can understand my listening habits.
9. As a user, I want the player to watch my music folders for file changes so that my library stays accurate without manual scans.
10. As a user, I want to export a playlist as a text/CSV file so that I can share it with others or open it somewhere else.
11. As a user, I want to update metadata (title, artist, etc.) so that I can fix potentially missing information.
12. As a user, I want to rate songs and view the rated songs in a playlist (playlist of tracks rated 10, 9, 8, etc.) so that I have a compilation of songs that I feel similarly about.
